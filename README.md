# CareerConnect - Job Board Website

CareerConnect is a responsive Job Board Website developed using HTML, CSS and JavaScript. The platform allows job seekers to explore opportunities, view job details, and apply for jobs while providing employers with a dashboard to manage job postings.

## Features

* Home Page with Featured Jobs
* Job Listings Page
* Job Details Page
* Candidate Dashboard
* Employer Dashboard
* Job Application Form
* Search Functionality
* Resume Upload Support
* Responsive Design
* Professional User Interface
